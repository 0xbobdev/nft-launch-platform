## Features

**Contract**
- ERC-721A for gas-efficient batch minting
- Whitelist phase with Merkle tree proof
- Per-wallet mint limit on public phase
- On-chain reveal with placeholder URI
- EIP-2981 royalty standard

**Frontend**
- Live mint counter · Whitelist checker
- Wallet connect with mobile support
- Post-mint gallery with reveal countdown

## Stack

- Solidity + ERC-721A · Hardhat · Merkle scripts · React · Wagmi · IPFS
